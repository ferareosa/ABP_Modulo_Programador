## ⚙️ Instrucciones de Uso

1. Clonar o descargar este repositorio.

2. Instalar las dependencias necesarias ejecutando el siguiente comando en la terminal:

   ```bash
   pip install .
   ```

3. Ejecutar el programa principal:

   ```bash
   python main.py
   ```
